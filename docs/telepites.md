# Telepítési lépések

## Előfeltételek

- Node.js 20+
- npm 10+

## Telepítés

```bash
npm install
cp starting-web/.env.example starting-web/.env
cp starting-app/.env.example starting-app/.env
```

## Indítás

```bash
npm run indit:web
npm run indit:app
```

## Ellenőrzés

```bash
npm run ellenoriz:web
npm run ellenoriz:app
npm run epit:web
```
